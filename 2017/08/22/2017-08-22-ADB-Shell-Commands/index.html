ADB:Android Debug Bridge 是能够和手机或模拟器通信的一个命令行工具。下载地址

命令分类

1. ADB Debugging

2. 连接设备

3. 包管理

adb install

各项参数可以结合使用

adb uninstall

删除一个应用包

adb shell pm list package

输出所有应用包名，可以追加选项过滤

选项命令可以结合使用

adb shell pm path

根据包名显示APK存放位置

adb shell pm clear

删除指定包名的所有数据，包括data和cache

4. 文件管理

adb pull

adb pull <remote> [local]

从设备下载文件到电脑

adb push

adb push <local> <remote>

从电脑复制文件到设备

adb shell ls

输出文件列表

  • ls 输出文件列表，不包含隐藏文件
  • ls -a 输出文件列表，包含隐藏文件
  • ls -i 输出文件列表，包含文件下标
  • ls -s 输出文件列表，包含文件的大小
  • ls -n 输出文件列表，包含文件的UID,GID
  • ls -R 递归列出子目录

adb shell rm

删除文件

  • rm -f 强制删除文件
  • rm -r 递归删除文件夹的文件
  • rm -d 删除文件夹,即使非空.(==rmdir)
  • rm -i 删除前提示

adb shell mkdir

创建目录

  • mkdir /sdcard/tmp
  • mkdir -m 777 /sdcard/tmp 创建并设置权限
  • mkdir -p /sdcard/tmp/sub1/sub2 根据需要创建父目录

adb shell touch 新建文件

adb shell pwd 返回当前路径

adb shell cp 复制文件

adb shell mv 移动或重命名文件

5. 网络

adb shell ip

ip [ OPTIONS ] OBJECT

6. Logcat

打印日子到屏幕

adb logcat [option] [filter-specs]

根据不同的级别打印日志，日志级别:V < D < I < W < E

  • V : Verbose 详细
  • D : Debug
  • I :Info
  • W :Warn
  • E : Error

adb logcat -b <Buffer>

查看缓冲区信息
